冷启动（硬约束）
匹配池没人 = 空刷 = 差体验。建档量是阶段二能否开启的硬门槛。
· 种子：帮会资源（45+ 帮主 / 管理）、KOC、官方红娘账号先行铺池
· 设单服 / 单市场最低建档阈值，未达标延后或合并区服开匹配
内容把控（公屏 = 公开广播）
宣言先过敏感词 / 外链 / 联系方式过滤再上墙；可延迟或抽样上墙、人工巡查；一人在墙数量限频；举报即下墙。
端内联动（可选 · 高价值）
精选宣言同步到游戏内场景（主城地标 / 灯会公屏弹幕），端内点击可唤起 H5，形成 H5×端内导流。

匹配池与权重
同 intent 同池，不跨轨。随机抽取 + 轻权重排序：标签重合度 > 同活跃时段 > 同区服；避免连续重复同一人。

· 被 ship 通知拉回，缩短异步错过

结果分层（人人有交代）
① 双向 ship = 红线自牵成（最高）：解锁私信 + 红线信物 + 一键组队进契约副本，专属标识。
② 单向 ship：你进入对方「心动池」，对方可在心动报告 / 被心动列表回 ship；你累计「我心动的人」。
③ 月老保底线：活动期内每人至少由系统撮合 1 条红线，可进契约副本，奖励低于自牵成。
④ 纯参与：swipe 满 N 次即得基础奖励 + 心动报告。
私信仍仅双向后开启（这是反骚扰，非难度）——契约副本 / 奖励 / 进度均不卡双向。
